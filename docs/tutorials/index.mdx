## LangGraph

LangGraph는 LLM을 사용하여 강력하고 상태가 있는 다중 액터 애플리케이션을 구축하기 위해 단계를 그래프의 엣지와 노드로 모델링하는 LangChain의 확장입니다.

LangGraph 문서는 현재 별도의 사이트에 호스팅되고 있습니다.
[여기에서 LangGraph 튜토리얼을 확인하세요](https://langchain-ai.github.io/langgraph/tutorials/).

## LangSmith

LangSmith는 LLM 애플리케이션을 면밀히 추적, 모니터링 및 평가할 수 있게 해줍니다.
LangChain과 원활하게 통합되며, 빌드하는 동안 체인의 개별 단계를 검사하고 디버그하는 데 사용할 수 있습니다.

LangSmith 문서는 별도의 사이트에 호스팅되고 있습니다.
[여기에서 LangSmith 튜토리얼을 확인하세요](https://docs.smith.langchain.com/tutorials/).

### 평가

LangSmith는 LLM 애플리케이션의 성능을 평가하는 데 도움을 줍니다. 아래 튜토리얼은 시작하기에 좋은 방법입니다:

- [LLM 애플리케이션 평가하기](https://docs.smith.langchain.com/tutorials/Developers/evaluation)
