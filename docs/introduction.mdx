# 소개

**LangChain**은 대형 언어 모델(LLM)로 구동되는 애플리케이션을 개발하기 위한 프레임워크입니다.

LangChain은 LLM 애플리케이션 생애 주기의 모든 단계를 간소화합니다:
- **개발**: LangChain의 오픈 소스 [구성 요소](/docs/concepts#langchain-expression-language-lcel), [컴포넌트](/docs/concepts), 및 [타사 통합](/docs/integrations/platforms/)을 사용하여 애플리케이션을 구축하세요. [LangGraph](/docs/concepts/#langgraph)를 사용하여 1급 스트리밍 및 인간 개입 지원이 있는 상태 저장 에이전트를 구축하세요.
- **생산화**: [LangSmith](https://docs.smith.langchain.com/)를 사용하여 체인을 검사, 모니터링 및 평가하여 지속적으로 최적화하고 자신 있게 배포할 수 있습니다.
- **배포**: LangGraph 애플리케이션을 [LangGraph Cloud](https://langchain-ai.github.io/langgraph/cloud/)를 사용하여 생산 준비가 완료된 API 및 어시스턴트로 변환하세요.

구체적으로, 이 프레임워크는 다음의 오픈 소스 라이브러리로 구성됩니다:

- **`langchain-core`**: 기본 추상화 및 LangChain 표현 언어.
- **`langchain-community`**: 타사 통합.
  - 파트너 패키지(예: **`langchain-openai`**, **`langchain-anthropic`** 등): 일부 통합은 **`langchain-core`**에만 의존하는 경량 패키지로 추가 분리되었습니다.
- **`langchain`**: 애플리케이션의 인지 아키텍처를 구성하는 체인, 에이전트 및 검색 전략.
- **[LangGraph](https://langchain-ai.github.io/langgraph)**: 단계를 그래프의 엣지와 노드로 모델링하여 LLM으로 강력하고 상태 저장의 다중 행위자 애플리케이션을 구축합니다. LangChain과 원활하게 통합되지만, 없이도 사용할 수 있습니다.
- **[LangServe](/docs/langserve)**: LangChain 체인을 REST API로 배포합니다.
- **[LangSmith](https://docs.smith.langchain.com)**: LLM 애플리케이션을 디버그, 테스트, 평가 및 모니터링할 수 있는 개발자 플랫폼입니다.

:::note

이 문서는 Python LangChain 라이브러리에 중점을 두고 있습니다. JavaScript LangChain 라이브러리에 대한 문서는 [여기](https://js.langchain.com)에서 확인하세요.

:::

## 생태계

### [🦜🛠️ LangSmith](https://docs.smith.langchain.com)
언어 모델 애플리케이션 및 지능형 에이전트를 추적하고 평가하여 프로토타입에서 생산으로 이동하는 데 도움을 줍니다.

### [🦜🕸️ LangGraph](https://langchain-ai.github.io/langgraph)
LLM으로 상태 저장의 다중 행위자 애플리케이션을 구축합니다. LangChain과 원활하게 통합되지만, 없이도 사용할 수 있습니다.
