# LLMs

:::caution
현재 [텍스트 완성 모델](/docs/concepts/#llms) 사용에 대한 문서 페이지에 있습니다. 최신의 가장 인기 있는 모델 중 많은 수가 [채팅 완성 모델](/docs/concepts/#chat-models)입니다.

더욱 고급 프롬프트 기법을 사용하고 있지 않다면, 아마도 [이 페이지](/docs/integrations/chat/)를 찾고 있을 것입니다.
:::

[LLMs](docs/concepts/#llms)는 문자열을 입력으로 받아 문자열을 출력하는 언어 모델입니다.

:::info

자신만의 LLM을 작성하고 싶다면 [이 방법](/docs/how_to/custom_llm/)을 참조하세요.
통합에 기여하고 싶다면 [통합 기여하기](/docs/contributing/integrations/)를 참조하세요.

:::
