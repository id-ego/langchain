# 앤트로픽

> [앤트로픽](https://www.anthropic.com/)은 AI 안전 및 연구 회사이며, `Claude`의 제작자입니다.
이 페이지는 `앤트로픽` 모델과 `LangChain` 간의 모든 통합을 다룹니다.

## 설치 및 설정

`앤트로픽` 모델을 사용하려면 파이썬 패키지를 설치해야 합니다:

```bash
pip install -U langchain-anthropic
```


`ANTHROPIC_API_KEY` 환경 변수를 설정해야 합니다.
앤트로픽 API 키는 [여기](https://console.anthropic.com/settings/keys)에서 얻을 수 있습니다.
