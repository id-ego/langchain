# 허깅 페이스

[허깅 페이스 플랫폼](https://huggingface.co/)과 관련된 모든 기능.

## 설치

대부분의 허깅 페이스 통합은 `langchain-huggingface` 패키지에서 사용할 수 있습니다.

```bash
pip install langchain-huggingface
```


## 채팅 모델

### 허깅 페이스의 모델

`허깅 페이스` LLM 클래스 또는 `ChatHuggingFace` 클래스를 직접 사용할 수 있습니다.

[사용 예시](/docs/integrations/chat/huggingface)를 참조하세요.

### HuggingFaceBgeEmbeddings

> [허깅 페이스의 BGE 모델](https://huggingface.co/BAAI/bge-large-en)은 [최고의 오픈 소스 임베딩 모델](https://huggingface.co/spaces/mteb/leaderboard)입니다.
BGE 모델은 [베이징 인공지능 연구원(BAAI)](https://en.wikipedia.org/wiki/Beijing_Academy_of_Artificial_Intelligence)에서 생성되었습니다. `BAAI`는 AI 연구 및 개발에 종사하는 비영리 민간 기관입니다.

[사용 예시](/docs/integrations/text_embedding/bge_huggingface)를 참조하세요.

```python
<!--IMPORTS:[{"imported": "HuggingFaceBgeEmbeddings", "source": "langchain_community.embeddings", "docs": "https://api.python.langchain.com/en/latest/embeddings/langchain_community.embeddings.huggingface.HuggingFaceBgeEmbeddings.html", "title": "Hugging Face"}]-->
```

### 허깅 페이스 텍스트 임베딩 추론 (TEI)

> [허깅 페이스 텍스트 임베딩 추론(TEI)](https://huggingface.co/docs/text-generation-inference/index)은 오픈 소스
텍스트 임베딩 및 시퀀스 분류 모델을 배포하고 제공하기 위한 도구 모음입니다. `TEI`는 `FlagEmbedding`, `Ember`, `GTE` 및 `E5`를 포함한 가장 인기 있는 모델에 대한 고성능 추출을 가능하게 합니다.

`huggingface-hub` 파이썬 패키지를 설치해야 합니다.

```bash
pip install huggingface-hub
```


[사용 예시](/docs/integrations/text_embedding/text_embeddings_inference)를 참조하세요.

```python
<!--IMPORTS:[{"imported": "HuggingFaceHubEmbeddings", "source": "langchain_community.embeddings", "docs": "https://api.python.langchain.com/en/latest/embeddings/langchain_community.embeddings.huggingface_hub.HuggingFaceHubEmbeddings.html", "title": "Hugging Face"}]-->
```

## 문서 로더

### 허깅 페이스 데이터셋

> [허깅 페이스 허브](https://huggingface.co/docs/hub/index)는 100개 이상의 언어로 75,000개 이상의
[데이터셋](https://huggingface.co/docs/hub/index#datasets)을 보유하고 있으며,
NLP, 컴퓨터 비전 및 오디오 전반에 걸쳐 다양한 작업에 사용할 수 있습니다.
이들은 번역, 자동 음성 인식 및 이미지 분류와 같은 다양한 작업에 사용됩니다.

`datasets` 파이썬 패키지를 설치해야 합니다.

```bash
pip install datasets
```


[사용 예시](/docs/integrations/document_loaders/hugging_face_dataset)를 참조하세요.
