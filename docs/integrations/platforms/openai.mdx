# OpenAI

OpenAI와 관련된 모든 기능

> [OpenAI](https://en.wikipedia.org/wiki/OpenAI)는 비영리 `OpenAI Incorporated`와 그 영리 자회사 `OpenAI Limited Partnership`로 구성된 미국의 인공지능(AI) 연구소입니다.  
`OpenAI`는 친근한 AI를 촉진하고 개발하기 위한 선언된 의도를 가지고 AI 연구를 수행합니다.  
`OpenAI` 시스템은 `Microsoft`의 `Azure` 기반 슈퍼컴퓨팅 플랫폼에서 실행됩니다.

> [OpenAI API](https://platform.openai.com/docs/models)는 다양한 기능과 가격대의 모델로 구동됩니다.  
>  
> [ChatGPT](https://chat.openai.com)는 `OpenAI`에서 개발한 인공지능(AI) 챗봇입니다.

## 설치 및 설정

통합 패키지를 설치하려면  
```bash
pip install langchain-openai
```


OpenAI API 키를 얻고 이를 환경 변수(`OPENAI_API_KEY`)로 설정합니다.

## 토크나이저

`tiktoken` 토크나이저를 사용할 수 있는 여러 장소가 있습니다. 기본적으로 OpenAI LLM의 토큰 수를 세는 데 사용됩니다.

문서를 분할할 때 토큰 수를 세는 데에도 사용할 수 있습니다.  
```python
<!--IMPORTS:[{"imported": "CharacterTextSplitter", "source": "langchain.text_splitter", "docs": "https://api.python.langchain.com/en/latest/character/langchain_text_splitters.character.CharacterTextSplitter.html", "title": "OpenAI"}]-->

CharacterTextSplitter.from_tiktoken_encoder(...)
```
  
이에 대한 보다 자세한 설명은 [이 노트북](/docs/how_to/split_by_token/#tiktoken)을 참조하세요.
