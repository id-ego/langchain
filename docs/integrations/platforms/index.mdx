# 공급자

:::info

자신만의 통합을 작성하고 싶다면 [LangChain 확장하기](/docs/how_to/#custom)를 참조하세요.  
통합에 기여하고 싶다면 [통합 기여하기](/docs/contributing/integrations/)를 참조하세요.

:::

LangChain은 많은 공급자와 통합됩니다.
