# 도구

[도구](/docs/concepts/#tools)는 모델에 의해 호출되도록 설계된 유틸리티입니다: 그들의 입력은 모델에 의해 생성되도록 설계되었고, 그들의 출력은 모델에 다시 전달되도록 설계되었습니다.

[툴킷](/docs/concepts#toolkits)은 함께 사용하기 위해 설계된 도구의 모음입니다.

:::info

자신만의 도구를 작성하고 싶다면 [이 방법](/docs/how_to/custom_tools/)을 참조하세요.
통합 기여를 원하신다면 [통합 기여하기](/docs/contributing/integrations/)를 참조하세요.

:::
