# 채팅 모델

[채팅 모델](/docs/concepts/#chat-models)은 일련의 [메시지](/docs/concepts/#messages)를 입력으로 사용하고 메시지를 출력으로 반환하는 언어 모델입니다(일반 텍스트를 사용하는 것과는 반대). 일반적으로 이러한 모델은 최신 모델입니다.

:::info

자신만의 채팅 모델을 작성하고 싶다면 [이 방법](/docs/how_to/custom_chat_model/)을 참조하세요. 통합에 기여하고 싶다면 [통합 기여하기](/docs/contributing/integrations/)를 참조하세요.

:::
