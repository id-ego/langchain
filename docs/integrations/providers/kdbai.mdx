# KDB.AI

> [KDB.AI](https://kdb.ai)는 고급 검색, 추천 및 개인화를 제공하여 실시간 데이터를 사용하여 확장 가능하고 신뢰할 수 있는 AI 애플리케이션을 구축할 수 있는 강력한 지식 기반 벡터 데이터베이스 및 검색 엔진입니다.

## 설치 및 설정

Python SDK를 설치하세요:

```bash
pip install kdbai-client
```
