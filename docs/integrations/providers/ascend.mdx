# Ascend

> [Ascend](https://https://www.hiascend.com/)는 Huawei에서 제공하는 자연 프로세스 유닛입니다.

이 페이지는 LangChain과 함께 ascend NPU를 사용하는 방법을 다룹니다.

### 설치

torch-npu를 사용하여 설치하려면:

```bash
pip install torch-npu
```


아래에 명시된 설치 지침을 따르십시오:
* CANN을 [여기](https://www.hiascend.com/document/detail/zh/canncommercial/700/quickstart/quickstart/quickstart_18_0002.html)와 같이 설치하십시오.
