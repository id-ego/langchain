# Trello

> [Trello](https://www.atlassian.com/software/trello)는 개인과 팀이 작업과 프로젝트를 조직하고 추적할 수 있도록 해주는 웹 기반 프로젝트 관리 및 협업 도구입니다. 사용자가 작업과 활동을 나타내기 위해 목록과 카드를 생성할 수 있는 "보드"라는 시각적 인터페이스를 제공합니다. TrelloLoader를 사용하면 `Trello` 보드에서 카드를 로드할 수 있습니다.

## 설치 및 설정

```bash
pip install py-trello beautifulsoup4
```
