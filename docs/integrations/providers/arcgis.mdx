# ArcGIS

> [ArcGIS](https://www.esri.com/en-us/arcgis/about-arcgis/overview)는 [Esri](https://www.esri.com/)에서 개발하고 유지 관리하는 클라이언트, 서버 및 온라인 지리 정보 시스템 소프트웨어의 모음입니다.
> 
> `ArcGISLoader`는 `arcgis` 패키지를 사용합니다.
`arcgis`는 벡터 및 래스터 분석, 지오코딩, 지도 제작, 경로 찾기 및 방향 설정을 위한 Python 라이브러리입니다. 
이 라이브러리는 GIS에서 사용자, 그룹 및 정보 항목을 관리하고 조직하며 관리합니다.
Esri 및 기타 권위 있는 출처의 즉시 사용 가능한 지도와 선별된 지리 데이터를 액세스할 수 있게 하며, 귀하의 데이터와도 함께 작업할 수 있습니다.

## 설치 및 설정

`arcgis` 패키지를 설치해야 합니다.

```bash
pip install -U arcgis
```
