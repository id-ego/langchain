# 화웨이

> [화웨이 기술 유한회사](https://www.huawei.com/)는 중국의 다국적 디지털 통신 기술 기업입니다.
> 
> [화웨이 클라우드](https://www.huaweicloud.com/intl/en-us/product/)는 포괄적인 글로벌 클라우드 컴퓨팅 서비스 모음을 제공합니다.

## 설치 및 설정

`화웨이 클라우드`에 접근하려면 액세스 토큰이 필요합니다.

또한 파이썬 라이브러리를 설치해야 합니다:

```bash
pip install -U esdk-obs-python
```
