# 아름다운 수프

> [아름다운 수프](https://www.crummy.com/software/BeautifulSoup/)는 HTML 및 XML 문서를 파싱하기 위한 파이썬 패키지입니다(형태가 잘못된 마크업, 즉 닫히지 않은 태그를 포함하여, 태그 수프(tag soup)라는 이름이 붙었습니다). 파싱된 페이지에 대한 파싱 트리를 생성하여 HTML에서 데이터를 추출하는 데 사용할 수 있으며,[3] 이는 웹 스크래핑에 유용합니다.

## 설치 및 설정

```bash
pip install beautifulsoup4
```
