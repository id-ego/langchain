# Arxiv

> [arXiv](https://arxiv.org/)는 물리학, 수학, 컴퓨터 과학, 정량 생물학, 정량 금융, 통계학, 전기 공학 및 시스템 과학, 경제학 분야의 200만 개 학술 기사를 위한 오픈 액세스 아카이브입니다.

## 설치 및 설정

먼저, `arxiv` 파이썬 패키지를 설치해야 합니다.

```bash
pip install arxiv
```


둘째, `arxiv.org` 사이트에서 다운로드한 PDF 파일을 텍스트 형식으로 변환하는 `PyMuPDF` 파이썬 패키지를 설치해야 합니다.

```bash
pip install pymupdf
```
