# SparkLLM

> [SparkLLM](https://xinghuo.xfyun.cn/spark)는 iFLYTEK에 의해 독립적으로 개발된 대규모 인지 모델입니다.  
방대한 양의 텍스트, 코드 및 이미지를 학습하여 교차 도메인 지식과 언어 이해 능력을 갖추고 있습니다.  
자연어 대화를 기반으로 작업을 이해하고 수행할 수 있습니다.
