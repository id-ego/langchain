# MindsDB

MindsDB는 기업 데이터를 기반으로 AI를 맞춤화하는 플랫폼입니다. MindsDB와 거의 200개의 [데이터 소스](https://docs.mindsdb.com/integrations/data-overview) 및 [AI/ML 프레임워크](https://docs.mindsdb.com/integrations/ai-overview)에 대한 통합을 통해, 모든 개발자는 자신의 목적에 맞게 기업 데이터를 사용하여 AI를 더 빠르고 안전하게 맞춤화할 수 있습니다.

MindsDB를 사용하면 모든 데이터 소스를 모든 AI/ML 모델에 연결하여 AI 기반 애플리케이션을 구현하고 자동화할 수 있습니다. 데이터베이스, 벡터 스토어 또는 애플리케이션의 데이터를 활용하여 모델을 실시간으로 배포, 제공 및 미세 조정할 수 있습니다. 모든 작업은 개발자가 이미 알고 있는 범용 도구를 사용하여 수행할 수 있습니다.

MindsDB는 LangChain과 통합되어 사용자가 다음을 수행할 수 있도록 합니다:

- MindsDB 내에서 LangChain을 통해 사용할 수 있는 모델을 배포하여 여러 데이터 소스에 접근할 수 있게 합니다.
- 실시간 및 동적 데이터를 사용하여 MindsDB 내에서 LangChain을 통해 사용할 수 있는 모델을 미세 조정합니다.
- LangChain과 MindsDB를 사용하여 AI 워크플로를 자동화합니다.

MindsDB의 LangChain 통합에 대해 자세히 알아보고 예제를 보려면 [우리 문서](https://docs.mindsdb.com/integrations/ai-engines/langchain)를 참조하세요.
