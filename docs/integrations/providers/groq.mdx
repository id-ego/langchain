# Groq

Groq에 오신 것을 환영합니다! 🚀 Groq에서는 세계 최초의 언어 처리 장치™(LPU)를 개발했습니다. Groq LPU는 결정론적이고 단일 코어 스트리밍 아키텍처를 가지고 있어 주어진 작업 부하에 대해 예측 가능하고 반복 가능한 성능으로 GenAI 추론 속도의 기준을 설정합니다.

아키텍처를 넘어, 우리의 소프트웨어는 여러분과 같은 개발자들이 혁신적이고 강력한 AI 애플리케이션을 만들 수 있도록 필요한 도구를 제공합니다. Groq를 엔진으로 사용하면 다음을 달성할 수 있습니다:

* 실시간 AI 및 HPC 추론을 위한 타협 없는 낮은 대기 시간과 성능을 달성 🔥
* 주어진 작업 부하에 대한 정확한 성능 및 컴퓨팅 시간을 알 수 있습니다 🔮
* 경쟁에서 앞서 나가기 위해 최첨단 기술을 활용할 수 있습니다 💪

더 많은 Groq를 원하십니까? [웹사이트](https://groq.com)에서 더 많은 자료를 확인하고, [Discord 커뮤니티](https://discord.gg/JvNsBDKeCG)에 참여하여 개발자들과 연결하세요!

## 설치 및 설정
통합 패키지를 설치합니다:

```bash
pip install langchain-groq
```


[API 키](https://wow.groq.com)를 요청하고 환경 변수로 설정합니다:

```bash
export GROQ_API_KEY=gsk_...
```
