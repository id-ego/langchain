# 미니맥스

> [미니맥스](https://api.minimax.chat)는 기업과 개인을 위한 자연어 처리 모델을 제공하는 중국 스타트업입니다.

## 설치 및 설정
[미니맥스 API 키](https://api.minimax.chat/user-center/basic-information/interface-key)를 얻고 이를 환경 변수(`MINIMAX_API_KEY`)로 설정하세요.  
[미니맥스 그룹 ID](https://api.minimax.chat/user-center/basic-information)를 얻고 이를 환경 변수(`MINIMAX_GROUP_ID`)로 설정하세요.

## LLM

미니맥스 LLM 래퍼가 존재하며, 이를 통해 접근할 수 있습니다.  
[사용 예시](/docs/integrations/llms/minimax)를 참조하세요.
