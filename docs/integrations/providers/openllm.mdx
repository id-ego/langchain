# OpenLLM

이 페이지는 LangChain과 함께 [OpenLLM](https://github.com/bentoml/OpenLLM)을 사용하는 방법을 보여줍니다.

`OpenLLM`은 대규모 언어 모델(LLM)을 운영하기 위한 오픈 플랫폼입니다. 개발자는 오픈 소스 LLM으로 쉽게 추론을 실행하고, 클라우드 또는 온프레미스에 배포하며, 강력한 AI 앱을 구축할 수 있습니다.

## 설치 및 설정

PyPI를 통해 OpenLLM 패키지를 설치합니다:

```bash
pip install openllm
```


## LLM

OpenLLM은 다양한 오픈 소스 LLM을 지원하며 사용자의 맞춤형 LLM도 제공합니다. `openllm model` 명령어를 사용하여 OpenLLM에 최적화된 모든 사용 가능한 모델을 확인하세요.

## 래퍼

LLM을 프로세스 내에서 로드하거나 원격 OpenLLM 서버에 접근할 수 있는 OpenLLM 래퍼가 있습니다:

```python
<!--IMPORTS:[{"imported": "OpenLLM", "source": "langchain_community.llms", "docs": "https://api.python.langchain.com/en/latest/llms/langchain_community.llms.openllm.OpenLLM.html", "title": "OpenLLM"}]-->
from langchain_community.llms import OpenLLM
```


### OpenLLM 서버용 래퍼

이 래퍼는 HTTP 또는 gRPC를 통해 OpenLLM 서버에 연결하는 것을 지원합니다. OpenLLM 서버는 로컬 또는 클라우드에서 실행할 수 있습니다.

로컬에서 시도하려면 OpenLLM 서버를 시작하세요:

```bash
openllm start flan-t5
```


래퍼 사용법:

```python
<!--IMPORTS:[{"imported": "OpenLLM", "source": "langchain_community.llms", "docs": "https://api.python.langchain.com/en/latest/llms/langchain_community.llms.openllm.OpenLLM.html", "title": "OpenLLM"}]-->
from langchain_community.llms import OpenLLM

llm = OpenLLM(server_url='http://localhost:3000')

llm("What is the difference between a duck and a goose? And why there are so many Goose in Canada?")
```


### 로컬 추론용 래퍼

OpenLLM 래퍼를 사용하여 현재 Python 프로세스에서 LLM을 로드하여 추론을 실행할 수도 있습니다.

```python
<!--IMPORTS:[{"imported": "OpenLLM", "source": "langchain_community.llms", "docs": "https://api.python.langchain.com/en/latest/llms/langchain_community.llms.openllm.OpenLLM.html", "title": "OpenLLM"}]-->
from langchain_community.llms import OpenLLM

llm = OpenLLM(model_name="dolly-v2", model_id='databricks/dolly-v2-7b')

llm("What is the difference between a duck and a goose? And why there are so many Goose in Canada?")
```
