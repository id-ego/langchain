# AssemblyAI

> [AssemblyAI](https://www.assemblyai.com/)는 음성-텍스트 변환, 화자 구분, 음성 요약 등과 같은 작업을 위한 `Speech AI` 모델을 구축합니다.
`AssemblyAI`의 Speech AI 모델은 음성 데이터(예: 통화, 가상 회의 및 팟캐스트)에 대한 정확한 음성-텍스트 변환, 화자 감지, 감정 분석, 장(chapter) 감지, PII(개인 식별 정보) 삭제를 포함합니다.

## 설치 및 설정

[API 키](https://www.assemblyai.com/dashboard/signup)를 받으세요.

`assemblyai` 패키지를 설치하세요.

```bash
pip install -U assemblyai
```
