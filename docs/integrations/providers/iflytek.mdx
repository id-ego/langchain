# iFlytek

> [iFlytek](https://www.iflytek.com)는 1999년에 설립된 중국의 정보 기술 회사입니다. 이 회사는 교육, 통신, 음악, 지능형 장난감 산업을 아우르는 음성 인식 소프트웨어 및 음성 기반 인터넷/모바일 제품을 만듭니다.

## 설치 및 설정

- [iFlyTek SparkLLM API Console](https://console.xfyun.cn/services/bm3)에서 `SparkLLM` app_id, api_key 및 api_secret을 가져옵니다 (자세한 내용은 [iFlyTek SparkLLM 소개](https://xinghuo.xfyun.cn/sparkapi)를 참조하세요).
- Python 패키지를 설치합니다 (임베딩 모델은 제외):

```bash
pip install websocket-client
```
