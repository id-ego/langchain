# IBM

`LangChain` 통합은 [IBM watsonx.ai](https://www.ibm.com/products/watsonx-ai) 플랫폼과 관련이 있습니다.

IBM® watsonx.ai™ AI 스튜디오는 IBM [watsonx](https://www.ibm.com/watsonx)™ AI 및 데이터 플랫폼의 일부로, [기초 모델](https://www.ibm.com/products/watsonx-ai/foundation-models)로 구동되는 새로운 생성 AI 기능과 전통적인 머신 러닝(ML)을 결합하여 AI 생애 주기를 아우르는 강력한 스튜디오를 제공합니다. 기업 데이터를 사용하여 모델을 조정하고 안내하여 요구 사항을 충족할 수 있도록 성능이 뛰어난 프롬프트를 구축하고 개선하기 위한 사용하기 쉬운 도구를 제공합니다. watsonx.ai를 사용하면 시간과 데이터의 일부만으로 AI 애플리케이션을 구축할 수 있습니다. Watsonx.ai는 다음을 제공합니다:

- **다양한 모델과 유연성:** IBM이 개발한 모델, 오픈 소스 및 타사 모델 중에서 선택하거나 자신만의 모델을 구축할 수 있습니다.
- **차별화된 클라이언트 보호:** IBM은 IBM이 개발한 모델을 지원하며, 클라이언트를 타사 IP 청구로부터 면책합니다.
- **종합적인 AI 거버넌스:** 기업은 비즈니스 전반에 걸쳐 신뢰할 수 있는 데이터를 사용하여 AI의 영향을 확장하고 가속화할 수 있습니다.
- **하이브리드, 멀티 클라우드 배포:** IBM은 선택한 하이브리드 클라우드 스택에 AI 작업을 통합하고 배포할 수 있는 유연성을 제공합니다.

## 설치 및 설정

통합 패키지를 설치하려면
```bash
pip install -qU langchain-ibm
```


IBM watsonx.ai API 키를 얻고 이를 환경 변수(`WATSONX_APIKEY`)로 설정합니다.
```python
import os

os.environ["WATSONX_APIKEY"] = "your IBM watsonx.ai api key"
```
