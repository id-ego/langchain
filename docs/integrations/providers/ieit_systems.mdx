# IEIT 시스템

> [IEIT 시스템](https://en.ieisystem.com/)은 1999년에 설립된 중국의 정보 기술 회사입니다. IT 인프라 제품, 솔루션 및 서비스, 클라우드 컴퓨팅, 빅 데이터 및 인공지능 분야의 혁신적인 IT 제품과 솔루션을 제공합니다.

```bash
pip install openai
```
