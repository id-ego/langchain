# Runhouse

이 페이지는 LangChain 내에서 [Runhouse](https://github.com/run-house/runhouse) 생태계를 사용하는 방법을 다룹니다. 설치 및 설정, LLM, 임베딩의 세 부분으로 나뉩니다.

## 설치 및 설정
- `pip install runhouse`로 Python SDK를 설치합니다.
- 온디맨드 클러스터를 사용하고 싶다면 `sky check`로 클라우드 자격 증명을 확인하세요.

## 자체 호스팅 LLM
기본적인 자체 호스팅 LLM을 위해 `SelfHostedHuggingFaceLLM` 클래스를 사용할 수 있습니다. 더 맞춤화된 LLM을 위해 `SelfHostedPipeline` 부모 클래스를 사용할 수 있습니다.

```python
<!--IMPORTS:[{"imported": "SelfHostedPipeline", "source": "langchain_community.llms", "docs": "https://api.python.langchain.com/en/latest/llms/langchain_community.llms.self_hosted.SelfHostedPipeline.html", "title": "Runhouse"}, {"imported": "SelfHostedHuggingFaceLLM", "source": "langchain_community.llms", "docs": "https://api.python.langchain.com/en/latest/llms/langchain_community.llms.self_hosted_hugging_face.SelfHostedHuggingFaceLLM.html", "title": "Runhouse"}]-->

```


자체 호스팅 LLM에 대한 더 자세한 안내는 [이 노트북](/docs/integrations/llms/runhouse)을 참조하세요.

## 자체 호스팅 임베딩

Hugging Face Transformers 모델에서 기본적인 자체 호스팅 임베딩을 위해 `SelfHostedEmbedding` 클래스를 사용할 수 있습니다.
