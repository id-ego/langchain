# Astra DB

> [DataStax Astra DB](https://docs.datastax.com/en/astra/home/astra.html)는 `Apache Cassandra®`를 기반으로 구축된 서버리스 벡터 기능을 갖춘 데이터베이스로, 사용하기 쉬운 JSON API를 통해 편리하게 제공됩니다.

[DataStax에서 제공하는 튜토리얼](https://docs.datastax.com/en/astra/astra-db-vector/tutorials/chatbot.html)을 참조하세요.

## 설치 및 설정

다음 Python 패키지를 설치하세요:
```bash
pip install "langchain-astradb>=0.1.0"
```


[연결 비밀](https://docs.datastax.com/en/astra/astra-db-vector/get-started/quickstart.html)을 가져옵니다.
다음 환경 변수를 설정하세요:

```python
ASTRA_DB_APPLICATION_TOKEN="TOKEN"
ASTRA_DB_API_ENDPOINT="API_ENDPOINT"
```
