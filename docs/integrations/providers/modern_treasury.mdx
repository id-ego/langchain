# 현대 재무

> [현대 재무](https://www.moderntreasury.com/)는 복잡한 결제 운영을 간소화합니다. 이는 자금을 이동시키는 제품과 프로세스를 지원하는 통합 플랫폼입니다.
> - 은행 및 결제 시스템에 연결
> - 실시간으로 거래 및 잔액 추적
> - 규모에 맞게 결제 운영 자동화
