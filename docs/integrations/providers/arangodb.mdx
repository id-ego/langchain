# ArangoDB

> [ArangoDB](https://github.com/arangodb/arangodb)는 연결된 데이터에서 가치를 더 빠르게 이끌어내기 위한 확장 가능한 그래프 데이터베이스 시스템입니다. 네이티브 그래프, 통합 검색 엔진 및 JSON 지원을 단일 쿼리 언어를 통해 제공합니다. ArangoDB는 온프레미스, 클라우드 등 어디에서나 실행됩니다.

## 설치 및 설정

다음 명령어로 [ArangoDB Python Driver](https://github.com/ArangoDB-Community/python-arango) 패키지를 설치하세요.

```bash
pip install python-arango
```
