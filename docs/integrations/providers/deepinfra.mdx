# DeepInfra

> [DeepInfra](https://deepinfra.com/docs)는 우리가 [최신 머신 러닝 모델](https://deepinfra.com/models)을 쉽게 실행할 수 있도록 해줍니다.  
DeepInfra는 모델 실행, 확장 및 모니터링과 관련된 모든 무거운 작업을 처리합니다. 사용자는 애플리케이션에 집중하고 간단한 REST API 호출로 모델을 통합할 수 있습니다.

> DeepInfra는 LangChain과의 통합에 대한 [예제](https://deepinfra.com/docs/advanced/langchain)를 제공합니다.

이 페이지는 `LangChain` 내에서 `DeepInfra` 생태계를 사용하는 방법을 다룹니다.  
설치 및 설정과 특정 DeepInfra 래퍼에 대한 참조의 두 부분으로 나뉩니다.

## 설치 및 설정

- 이 링크 [여기](https://deepinfra.com/)에서 DeepInfra API 키를 가져옵니다.  
- DeepInfra API 키를 가져와서 환경 변수(`DEEPINFRA_API_TOKEN`)로 설정합니다.

## 사용 가능한 모델

DeepInfra는 배포 준비가 완료된 다양한 오픈 소스 LLM을 제공합니다.

[text-generation](https://deepinfra.com/models?type=text-generation) 및  
[embeddings](https://deepinfra.com/models?type=embeddings)에 대한 지원 모델을 확인할 수 있습니다.

[요청 및 응답 매개변수 목록](https://deepinfra.com/meta-llama/Llama-2-70b-chat-hf/api)을 볼 수 있습니다.

채팅 모델은 [openai api](https://deepinfra.com/meta-llama/Llama-2-70b-chat-hf/api?example=openai-http)를 따릅니다.
