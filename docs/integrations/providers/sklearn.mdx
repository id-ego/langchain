# scikit-learn

> [scikit-learn](https://scikit-learn.org/stable/)은 기계 학습 알고리즘의 오픈 소스 모음으로, [k 최근접 이웃](https://scikit-learn.org/stable/modules/generated/sklearn.neighbors.NearestNeighbors.html)의 일부 구현을 포함합니다. `SKLearnVectorStore`는 이 구현을 감싸고 벡터 저장소를 json, bson(이진 json) 또는 Apache Parquet 형식으로 지속할 수 있는 기능을 추가합니다.

## 설치 및 설정

- `pip install scikit-learn`로 Python 패키지를 설치합니다.

## 벡터 저장소

`SKLearnVectorStore`는 scikit-learn 패키지의 최근접 이웃 구현을 간단하게 감싸며, 이를 벡터 저장소로 사용할 수 있게 해줍니다.

이 벡터 저장소를 가져오려면:
