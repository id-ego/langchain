# Zhipu AI

> [Zhipu AI](https://www.zhipuai.cn/en/aboutus)는 `칭화대학교 컴퓨터 과학과`의 기술 발전에서 유래된 인공지능 회사로, 기계가 인간처럼 사고하도록 가르치는 사명을 가지고 있습니다. 세계 최고의 AI 팀은 최첨단 대형 언어 및 다중 모달 모델을 개발하고, 고정밀의 수십억 규모 지식 그래프를 구축하였으며, 이 조합은 인공지능 일반 지능을 향한 강력한 데이터 및 지식 기반 인지 엔진을 만드는 데 독특한 힘을 제공합니다.
