# Activeloop Deep Lake

> [Activeloop Deep Lake](https://docs.activeloop.ai/)는 딥 러닝 애플리케이션을 위한 데이터 레이크로, 벡터 저장소로 사용할 수 있습니다.

## 왜 Deep Lake인가요?

- 단순한 (다중 모달) 벡터 저장소 이상입니다. 나중에 데이터셋을 사용하여 자신의 LLM 모델을 미세 조정할 수 있습니다.
- 임베딩뿐만 아니라 원본 데이터도 자동 버전 관리와 함께 저장합니다.
- 진정한 서버리스입니다. 다른 서비스를 요구하지 않으며 주요 클라우드 제공업체(`AWS S3`, `GCS` 등)와 함께 사용할 수 있습니다.

`Activeloop Deep Lake`는 `SelfQuery Retrieval`을 지원합니다:
[Activeloop Deep Lake Self Query Retrieval](/docs/integrations/retrievers/self_query/activeloop_deeplake_self_query)
