# Kinetica

[Kinetica](https://www.kinetica.com/)는 시계열 및 공간 데이터에 대한 분석 및 생성 AI를 가능하게 하기 위해 특별히 제작된 실시간 데이터베이스입니다.

## Chat Model

Kinetica LLM 래퍼는 자연어를 SQL로 변환하여 데이터 검색 프로세스를 단순화하기 위해 [Kinetica SqlAssist LLM](https://docs.kinetica.com/7.2/sql-gpt/concepts/)을 사용합니다.

사용법은 [Kinetica Language To SQL Chat Model](/docs/integrations/chat/kinetica)를 참조하세요.

```python
<!--IMPORTS:[{"imported": "ChatKinetica", "source": "langchain_community.chat_models.kinetica", "docs": "https://api.python.langchain.com/en/latest/chat_models/langchain_community.chat_models.kinetica.ChatKinetica.html", "title": "Kinetica"}]-->
from langchain_community.chat_models.kinetica import ChatKinetica
```

사용법은 [Kinetica Vectorsore API](/docs/integrations/vectorstores/kinetica)를 참조하세요.

```python
<!--IMPORTS:[{"imported": "Kinetica", "source": "langchain_community.vectorstores", "docs": "https://api.python.langchain.com/en/latest/vectorstores/langchain_community.vectorstores.kinetica.Kinetica.html", "title": "Kinetica"}]-->
from langchain_community.vectorstores import Kinetica
```

## Retriever

Kinetica 리트리버는 비구조적 쿼리를 기반으로 문서를 반환할 수 있습니다.

사용법은 [Kinetica VectorStore based Retriever](/docs/integrations/retrievers/kinetica)를 참조하세요.
