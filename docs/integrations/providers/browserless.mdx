# 브라우저리스

> [브라우저리스](https://www.browserless.io/docs/start)는 클라우드에서 헤드리스 Chrome 인스턴스를 실행할 수 있는 서비스입니다. 자체 인프라를 관리하는 것에 대한 걱정 없이 대규모로 브라우저 기반 자동화를 실행하는 훌륭한 방법입니다.

## 설치 및 설정

API 키를 [여기](https://www.browserless.io/pricing/)에서 받아야 합니다.
