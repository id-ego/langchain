# Momento

> [Momento Cache](https://docs.momentohq.com/)는 세계 최초의 진정한 서버리스 캐싱 서비스로, 즉각적인 탄력성, 제로 스케일링 기능 및 빠른 성능을 제공합니다.
> 
> [Momento Vector Index](https://docs.momentohq.com/vector-index)는 가장 생산적이고 사용하기 쉬운 완전한 서버리스 벡터 인덱스로 두드러집니다.
> 
> 두 서비스 모두 SDK를 가져오고, API 키를 얻고, 코드에 몇 줄을 입력하면 준비가 완료됩니다. 이들은 함께 LLM 데이터 요구 사항에 대한 포괄적인 솔루션을 제공합니다.

이 페이지에서는 LangChain 내에서 [Momento](https://gomomento.com) 생태계를 사용하는 방법을 다룹니다.

## 설치 및 설정

- [여기](https://console.gomomento.com/)에서 무료 계정을 등록하여 API 키를 받으세요.
- `pip install momento`로 Momento Python SDK를 설치하세요.

## 캐시

Momento를 서버리스, 분산, 저지연 캐시로 사용하여 LLM 프롬프트와 응답을 처리하세요. 표준 캐시는 모든 환경에서 Momento 사용자의 주요 사용 사례입니다.

Momento Cache를 애플리케이션에 통합하려면:

```python
<!--IMPORTS:[{"imported": "MomentoCache", "source": "langchain.cache", "docs": "https://api.python.langchain.com/en/latest/cache/langchain_community.cache.MomentoCache.html", "title": "Momento"}]-->
from langchain.cache import MomentoCache
```


그런 다음 다음 코드를 사용하여 설정하세요:

```python
<!--IMPORTS:[{"imported": "set_llm_cache", "source": "langchain.globals", "docs": "https://api.python.langchain.com/en/latest/globals/langchain.globals.set_llm_cache.html", "title": "Momento"}]-->
from datetime import timedelta
from momento import CacheClient, Configurations, CredentialProvider
from langchain.globals import set_llm_cache

# Instantiate the Momento client
cache_client = CacheClient(
    Configurations.Laptop.v1(),
    CredentialProvider.from_environment_variable("MOMENTO_API_KEY"),
    default_ttl=timedelta(days=1))

# Choose a Momento cache name of your choice
cache_name = "langchain"

# Instantiate the LLM cache
set_llm_cache(MomentoCache(cache_client, cache_name))
```
