# Chaindesk

> [Chaindesk](https://chaindesk.ai)는 개인 데이터를 대형 언어 모델과 연결하는 데 도움을 주는 [오픈 소스](https://github.com/gmpetrov/databerry) 문서 검색 플랫폼입니다.

## 설치 및 설정

Chaindesk에 가입하고, 데이터 저장소를 생성하고, 데이터를 추가한 다음 데이터 저장소 API 엔드포인트 URL을 얻어야 합니다. [API 키](https://docs.chaindesk.ai/api-reference/authentication)가 필요합니다.
