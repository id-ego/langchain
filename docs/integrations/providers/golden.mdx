# 골든

> [골든](https://golden.com)은 골든 지식 그래프를 사용하여 쿼리 및 보강을 위한 자연어 API 세트를 제공합니다. 예를 들어 `OpenAI의 제품`, `시리즈 A 자금을 받은 생성 AI 회사`, `투자하는 래퍼`와 같은 쿼리를 사용하여 관련 엔티티에 대한 구조화된 데이터를 검색할 수 있습니다.
> 
> `golden-query` langchain 도구는 이러한 결과에 대한 프로그래밍적 접근을 가능하게 하는 [골든 쿼리 API](https://docs.golden.com/reference/query-api) 위에 래핑된 도구입니다.
자세한 내용은 [골든 쿼리 API 문서](https://docs.golden.com/reference/query-api)를 참조하세요.

## 설치 및 설정
- [골든 API 문서](https://docs.golden.com/)로 가서 골든 API에 대한 개요를 확인하세요.
- [골든 API 설정](https://golden.com/settings/api) 페이지에서 API 키를 받으세요.
- API 키를 GOLDEN_API_KEY 환경 변수에 저장하세요.

## 래퍼

### 유틸리티

이 API를 래핑하는 GoldenQueryAPIWrapper 유틸리티가 존재합니다. 이 유틸리티를 가져오려면:

```python
<!--IMPORTS:[{"imported": "GoldenQueryAPIWrapper", "source": "langchain_community.utilities.golden_query", "docs": "https://api.python.langchain.com/en/latest/utilities/langchain_community.utilities.golden_query.GoldenQueryAPIWrapper.html", "title": "Golden"}]-->
from langchain_community.utilities.golden_query import GoldenQueryAPIWrapper
```


이 래퍼에 대한 더 자세한 설명은 [이 노트북](/docs/integrations/tools/golden_query)을 참조하세요.

### 도구

이 래퍼를 도구로 쉽게 로드하여 에이전트와 함께 사용할 수 있습니다.
다음과 같이 할 수 있습니다:
```python
<!--IMPORTS:[{"imported": "load_tools", "source": "langchain.agents", "docs": "https://api.python.langchain.com/en/latest/agent_toolkits/langchain_community.agent_toolkits.load_tools.load_tools.html", "title": "Golden"}]-->

```


도구에 대한 더 많은 정보는 [이 페이지](/docs/how_to/tools_builtin)를 참조하세요.
