# 아틀라스

> [Nomic Atlas](https://docs.nomic.ai/index.html)는 소규모 및 인터넷 규모의 비정형 데이터셋과 상호작용하기 위한 플랫폼입니다.

## 설치 및 설정

- `pip install nomic`를 사용하여 Python 패키지를 설치합니다.
- `Nomic`은 langchains의 poetry extras `poetry install -E all`에도 포함되어 있습니다.
