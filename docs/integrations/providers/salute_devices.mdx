# Salute Devices

Salute Devices는 GigaChat LLM의 모델을 제공합니다.

GigaChat에 접근하는 방법에 대한 자세한 정보는 [여기를 클릭하세요](https://developers.sber.ru/docs/ru/gigachat/api/integration).

## 설치 및 설정

GigaChat 패키지는 PyPI에서 pip를 통해 설치할 수 있습니다:

```bash
pip install gigachat
```
