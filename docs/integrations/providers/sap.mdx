# SAP

> [SAP SE(Wikipedia)](https://www.sap.com/about/company.html)는 독일의 다국적 소프트웨어 회사입니다. 이 회사는 비즈니스 운영 및 고객 관계 관리를 위한 기업 소프트웨어를 개발합니다. 이 회사는 세계 최고의 `기업 자원 계획 (ERP)` 소프트웨어 공급업체입니다.

## 설치 및 설정

`hdbcli` 파이썬 패키지를 설치해야 합니다.

```bash
pip install hdbcli
```
