# MediaWikiDump

> [MediaWiki XML 덤프](https://www.mediawiki.org/wiki/Manual:Importing_XML_dumps)는 사이트 관련 데이터 없이 위키의 내용을 포함합니다
(모든 수정 사항이 포함된 위키 페이지). XML 덤프는 위키 데이터베이스의 전체 백업을 생성하지 않으며, 덤프에는 사용자 계정, 이미지, 편집 로그 등이 포함되지 않습니다.

## 설치 및 설정

여러 개의 파이썬 패키지를 설치해야 합니다.

`mediawiki-utilities`는 병합되지 않은 브랜치에서 XML 스키마 0.11을 지원합니다.
```bash
pip install -qU git+https://github.com/mediawiki-utilities/python-mwtypes@updates_schema_0.11
```


`mediawiki-utilities mwxml`에는 버그가 있으며, 수정 PR이 대기 중입니다.

```bash
pip install -qU git+https://github.com/gdedrouas/python-mwxml@xml_format_0.11
pip install -qU mwparserfromhell
```
