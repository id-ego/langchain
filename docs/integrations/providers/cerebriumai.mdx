# CerebriumAI

> [Cerebrium](https://docs.cerebrium.ai/cerebrium/getting-started/introduction)은 서버리스 GPU 인프라 제공업체입니다.  
여러 LLM 모델에 대한 API 액세스를 제공합니다.

[CerebriumAI 문서](https://docs.cerebrium.ai/examples/langchain)에서 예제를 확인하세요.

## 설치 및 설정

- 파이썬 패키지를 설치하세요:
```bash
pip install cerebrium
```


- [CerebriumAI API 키를 받으세요](https://docs.cerebrium.ai/cerebrium/getting-started/installation) 그리고  
환경 변수(`CEREBRIUMAI_API_KEY`)로 설정하세요.
