# 01.AI

> [01.AI](https://www.lingyiwanwu.com/en)는 Dr. Kai-Fu Lee가 설립한 AI 2.0의 최전선에 있는 글로벌 회사입니다. 그들은 6B에서 수백억 개의 매개변수에 이르는 Yi 시리즈를 포함한 최첨단 대형 언어 모델을 제공합니다. 01.AI는 또한 다중 모드 모델, 오픈 API 플랫폼 및 Yi-34B/9B/6B 및 Yi-VL과 같은 오픈 소스 옵션을 제공합니다.
