# Motherduck

> [Motherduck](https://motherduck.com/)는 관리형 DuckDB 클라우드 서비스입니다.

## 설치 및 설정

먼저, `duckdb` 파이썬 패키지를 설치해야 합니다.

```bash
pip install duckdb
```


또한 [Motherduck](https://motherduck.com/)에서 계정을 등록해야 합니다.

그 후, 연결 문자열을 설정해야 합니다. 우리는 주로 SQLAlchemy를 통해 Motherduck과 통합합니다.
연결 문자열은 다음과 같은 형식일 가능성이 높습니다:

```
token="..."

conn_str = f"duckdb:///md:{token}@my_db"
```


## SQLChain

SQLChain을 사용하여 자연어로 Motherduck 인스턴스의 데이터를 쿼리할 수 있습니다.

```
from langchain_openai import OpenAI
from langchain_community.utilities import SQLDatabase
from langchain_experimental.sql import SQLDatabaseChain
db = SQLDatabase.from_uri(conn_str)
db_chain = SQLDatabaseChain.from_llm(OpenAI(temperature=0), db, verbose=True)
```


여기에서 [SQL Chain](/docs/how_to#qa-over-sql--csv) 문서를 참조하여 사용하는 방법을 확인하세요.

## LLMCache

Motherduck을 사용하여 LLM 요청을 쉽게 캐시할 수도 있습니다.
다시 말해, 이는 SQLAlchemy 래퍼를 통해 수행됩니다.

```
import sqlalchemy
from langchain.globals import set_llm_cache
eng = sqlalchemy.create_engine(conn_str)
set_llm_cache(SQLAlchemyCache(engine=eng))
```


여기에서 [LLM Caching](/docs/integrations/llm_caching) 문서를 참조하여 사용하는 방법을 확인하세요.
