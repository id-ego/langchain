# 위키백과

> [위키백과](https://wikipedia.org/)는 위키피디언으로 알려진 자원봉사자 커뮤니티에 의해 작성되고 유지되는 다국어 무료 온라인 백과사전으로, 열린 협업과 MediaWiki라는 위키 기반 편집 시스템을 사용합니다. `위키백과`는 역사상 가장 크고 가장 많이 읽힌 참고 자료입니다.

## 설치 및 설정

```bash
pip install wikipedia
```
