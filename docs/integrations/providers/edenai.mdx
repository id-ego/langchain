# 에덴 AI

> [에덴 AI](https://docs.edenai.co/docs/getting-started-with-eden-ai) 사용자 인터페이스(UI)는 AI 프로젝트를 처리하기 위해 설계되었습니다. `에덴 AI 포털`을 사용하면 시장에서 최고의 엔진을 사용하여 코드 없이 AI를 수행할 수 있습니다.

## 설치 및 설정

에덴 AI API에 접근하려면 API 키가 필요하며, 이는 [계정 생성](https://app.edenai.run/user/register)을 통해 얻을 수 있으며, [여기](https://app.edenai.run/admin/account/settings)로 이동하면 됩니다.

## 도구

에덴 AI는 에이전트가 다음과 같은 여러 작업을 수행할 수 있는 도구 목록을 제공합니다:
* 음성 인식
* 텍스트 음성 변환
* 텍스트 명시적 콘텐츠 탐지
* 이미지 명시적 콘텐츠 탐지
* 객체 탐지
* OCR 송장 파싱
* OCR 신분증 파싱
