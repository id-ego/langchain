# 네뷸라

이 페이지는 [네뷸라](https://symbl.ai/nebula), [Symbl.ai](https://symbl.ai/)의 LLM 생태계를 LangChain 내에서 사용하는 방법을 다룹니다. 설치 및 설정과 특정 네뷸라 래퍼에 대한 참조로 두 부분으로 나뉩니다.

## 설치 및 설정

- [네뷸라 API 키](https://info.symbl.ai/Nebula_Private_Beta.html)를 얻고 환경 변수 `NEBULA_API_KEY`로 설정하세요.
- 더 자세한 내용은 [네뷸라 문서](https://docs.symbl.ai/docs/nebula-llm)를 참조하세요.

### LLM

네뷸라 LLM 래퍼가 존재하며, 이를 통해 접근할 수 있습니다.
```python
<!--IMPORTS:[{"imported": "Nebula", "source": "langchain_community.llms", "docs": "https://api.python.langchain.com/en/latest/llms/langchain_community.llms.symblai_nebula.Nebula.html", "title": "Nebula"}]-->

llm = Nebula()
```
