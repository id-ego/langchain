# GooseAI

이 페이지는 LangChain 내에서 GooseAI 생태계를 사용하는 방법을 다룹니다. 설치 및 설정과 특정 GooseAI 래퍼에 대한 참조로 나뉩니다.

## 설치 및 설정
- `pip install openai`로 Python SDK를 설치합니다.
- 이 링크 [여기](https://goose.ai/)에서 GooseAI API 키를 가져옵니다.
- 환경 변수(`GOOSEAI_API_KEY`)를 설정합니다.

```python
import os
os.environ["GOOSEAI_API_KEY"] = "YOUR_API_KEY"
```
