# Johnsnowlabs

[Johnsnowlabs](https://www.johnsnowlabs.com/)의 기업 NLP 라이브러리 생태계에 접근하여 200개 이상의 언어로 21,000개 이상의 기업 NLP 모델을 제공하는 오픈 소스 `johnsnowlabs` 라이브러리를 이용하세요. 24,000개 이상의 모델에 대한 자세한 내용은 [John Snow Labs 모델 허브](https://nlp.johnsnowlabs.com/models)를 참조하세요.

## 설치 및 설정

```bash
pip install johnsnowlabs
```


[기업 기능 설치](https://nlp.johnsnowlabs.com/docs/en/jsl/install_licensed_quick)를 위해 다음을 실행하세요:
```python
# for more details see https://nlp.johnsnowlabs.com/docs/en/jsl/install_licensed_quick
nlp.install()
```


쿼리와 문서를 `gpu`, `cpu`, `apple_silicon`, `aarch` 기반 최적화된 바이너리로 임베드할 수 있습니다. 기본적으로 cpu 바이너리가 사용됩니다. 세션이 시작되면 GPU 또는 CPU 간 전환을 위해 노트북을 재시작해야 하며, 그렇지 않으면 변경 사항이 적용되지 않습니다.

## CPU로 쿼리 임베드:
```python
document = "foo bar"
embedding = JohnSnowLabsEmbeddings('embed_sentence.bert')
output = embedding.embed_query(document)
```


## GPU로 쿼리 임베드:

```python
document = "foo bar"
embedding = JohnSnowLabsEmbeddings('embed_sentence.bert','gpu')
output = embedding.embed_query(document)
```


## Apple Silicon (M1, M2 등)으로 쿼리 임베드:

```python
documents = ["foo bar", 'bar foo']
embedding = JohnSnowLabsEmbeddings('embed_sentence.bert','apple_silicon')
output = embedding.embed_query(document)
```


## AARCH로 쿼리 임베드:

```python
documents = ["foo bar", 'bar foo']
embedding = JohnSnowLabsEmbeddings('embed_sentence.bert','aarch')
output = embedding.embed_query(document)
```


## CPU로 문서 임베드:
```python
documents = ["foo bar", 'bar foo']
embedding = JohnSnowLabsEmbeddings('embed_sentence.bert','gpu')
output = embedding.embed_documents(documents)
```

documents = ["foo bar", 'bar foo']
