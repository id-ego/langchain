# SingleStoreDB

> [SingleStoreDB](https://singlestore.com/)는 클라우드([cloud](https://www.singlestore.com/cloud/))와 온프레미스 모두에서 배포를 지원하는 고성능 분산 SQL 데이터베이스입니다. 벡터 저장소와 [dot_product](https://docs.singlestore.com/managed-service/en/reference/sql-reference/vector-functions/dot_product.html), [euclidean_distance](https://docs.singlestore.com/managed-service/en/reference/sql-reference/vector-functions/euclidean_distance.html)와 같은 벡터 기능을 제공하여 텍스트 유사성 매칭이 필요한 AI 애플리케이션을 지원합니다.

## 설치 및 설정

데이터베이스에 [연결](https://singlestoredb-python.labs.singlestore.com/generated/singlestoredb.connect.html)을 설정하는 방법에는 여러 가지가 있습니다. 환경 변수를 설정하거나 `SingleStoreDB constructor`에 이름이 지정된 매개변수를 전달할 수 있습니다. 또는 이러한 매개변수를 `from_documents` 및 `from_texts` 메서드에 제공할 수 있습니다.

```bash
pip install singlestoredb
```
