# 오라클 클라우드 인프라스트럭처 (OCI)

`LangChain` 통합과 관련된 [오라클 클라우드 인프라스트럭처](https://www.oracle.com/artificial-intelligence/)입니다.

## OCI 생성 AI
> 오라클 클라우드 인프라스트럭처 (OCI) [생성 AI](https://docs.oracle.com/en-us/iaas/Content/generative-ai/home.htm)는 최첨단의 사용자 정의 가능한 대형 언어 모델(LLM) 세트를 제공하는 완전 관리형 서비스로, 광범위한 사용 사례를 포괄하며 단일 API를 통해 제공됩니다. OCI 생성 AI 서비스를 사용하면 즉시 사용할 수 있는 사전 훈련된 모델에 접근하거나, 전용 AI 클러스터에서 자신의 데이터에 기반하여 세밀하게 조정된 사용자 정의 모델을 생성하고 호스팅할 수 있습니다.

사용하려면 최신 `oci` 파이썬 SDK와 langchain_community 패키지가 설치되어 있어야 합니다.

```bash
pip install -U oci langchain-community
```


[채팅](/docs/integrations/llms/oci_generative_ai), [완료](/docs/integrations/chat/oci_generative_ai), 및 [임베딩](/docs/integrations/text_embedding/oci_generative_ai) 사용 예제를 참조하세요.

## OCI 데이터 과학 모델 배포 엔드포인트

> [OCI 데이터 과학](https://docs.oracle.com/en-us/iaas/data-science/using/home.htm)은 데이터 과학 팀을 위한 완전 관리형 서버리스 플랫폼입니다. OCI 데이터 과학 플랫폼을 사용하면 기계 학습 모델을 구축, 훈련 및 관리한 후, [OCI 데이터 과학 모델 배포 서비스](https://docs.oracle.com/en-us/iaas/data-science/using/model-dep-about.htm)를 사용하여 OCI 모델 배포 엔드포인트로 배포할 수 있습니다.

VLLM 또는 TGI 프레임워크로 LLM을 배포한 경우, `OCIModelDeploymentVLLM` 또는 `OCIModelDeploymentTGI` 클래스를 사용하여 상호작용할 수 있습니다.

사용하려면 최신 `oracle-ads` 파이썬 SDK가 설치되어 있어야 합니다.

```bash
pip install -U oracle-ads
```


[사용 예제](/docs/integrations/llms/oci_model_deployment_endpoint)를 참조하세요.
