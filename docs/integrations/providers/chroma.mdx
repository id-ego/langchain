# Chroma

> [Chroma](https://docs.trychroma.com/getting-started)는 임베딩을 사용하여 AI 애플리케이션을 구축하기 위한 데이터베이스입니다.

## 설치 및 설정

```bash
pip install langchain-chroma
```
