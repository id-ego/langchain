# PromptLayer

> [PromptLayer](https://docs.promptlayer.com/introduction)는 프롬프트 엔지니어링을 위한 플랫폼입니다.  
또한 요청을 시각화하고, 프롬프트 버전을 관리하며, 사용량을 추적하는 LLM 가시성을 지원합니다.  
>  
`PromptLayer`는 LangChain과 직접 통합되는 LLM을 가지고 있지만(e.g.  
[`PromptLayerOpenAI`](https://docs.promptlayer.com/languages/langchain)),  
콜백을 사용하는 것이 `PromptLayer`를 LangChain과 통합하는 권장 방법입니다.

## 설치 및 설정

`PromptLayer`와 작업하기 위해서는 다음을 수행해야 합니다:  
- `PromptLayer` 계정 생성  
- API 토큰 생성 및 환경 변수로 설정 (`PROMPTLAYER_API_KEY`)

Python 패키지 설치:

```bash
pip install promptlayer
```
