# Postgres Embedding

> [pg_embedding](https://github.com/neondatabase/pg_embedding)은 `Postgres`와 `Hierarchical Navigable Small Worlds` 알고리즘을 사용하여 근사 최근접 이웃 검색을 위한 벡터 유사성 검색을 위한 오픈 소스 패키지입니다.

## Installation and Setup

여러 개의 파이썬 패키지를 설치해야 합니다.

```bash
pip install psycopg2-binary
```
