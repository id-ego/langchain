# PipelineAI

이 페이지는 LangChain 내에서 PipelineAI 생태계를 사용하는 방법을 다룹니다. 설치 및 설정과 특정 PipelineAI 래퍼에 대한 참조로 두 부분으로 나뉩니다.

## 설치 및 설정

- `pip install pipeline-ai`로 설치
- Pipeline Cloud API 키를 얻고 이를 환경 변수(`PIPELINE_API_KEY`)로 설정
