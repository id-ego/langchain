# CTranslate2

> [CTranslate2](https://opennmt.net/CTranslate2/quickstart.html)는 Transformer 모델을 위한 효율적인 추론을 위한 C++ 및 Python 라이브러리입니다.
> 
> 이 프로젝트는 CPU 및 GPU에서 Transformer 모델의 성능을 가속화하고 메모리 사용량을 줄이기 위해 가중치 양자화, 레이어 융합, 배치 재정렬 등과 같은 많은 성능 최적화 기술을 적용하는 맞춤형 런타임을 구현합니다.
> 
> 전체 기능 목록 및 지원되는 모델은 [프로젝트의 저장소](https://opennmt.net/CTranslate2/guides/transformers.html)에 포함되어 있습니다.
시작하려면 공식 [빠른 시작 가이드](https://opennmt.net/CTranslate2/quickstart.html)를 확인하세요.

## 설치 및 설정

Python 패키지를 설치하세요:

```bash
pip install ctranslate2
```
