# MLX

> [MLX](https://ml-explore.github.io/mlx/build/html/index.html)는 `Apple` 실리콘에서 효율적이고 유연한 머신 러닝을 위해 설계된 `NumPy`-유사 배열 프레임워크로, `Apple 머신 러닝 연구`에서 제공합니다.

## 설치 및 설정

여러 Python 패키지를 설치합니다:

```bash
pip install mlx-lm transformers huggingface_hub
```
