# 랜턴

이 페이지는 LangChain 내에서 [랜턴](https://github.com/lanterndata/lantern)을 사용하는 방법을 다룹니다. 
설정과 특정 랜턴 래퍼에 대한 참조의 두 부분으로 나뉩니다.

## 설정
1. 첫 번째 단계는 `lantern` 확장이 설치된 데이터베이스를 만드는 것입니다.
   
   데이터베이스와 확장을 설치하려면 [랜턴 설치 가이드](https://github.com/lanterndata/lantern#-quick-install)의 단계를 따르세요. 도커 이미지는 시작하는 가장 쉬운 방법입니다.

## 래퍼

### 벡터스토어

Postgres 벡터 데이터베이스를 감싸는 래퍼가 존재하여, 의미 검색이나 예제 선택을 위해 벡터스토어로 사용할 수 있습니다.

이 벡터스토어를 가져오려면:
```python
<!--IMPORTS:[{"imported": "Lantern", "source": "langchain_community.vectorstores", "docs": "https://api.python.langchain.com/en/latest/vectorstores/langchain_community.vectorstores.lantern.Lantern.html", "title": "Lantern"}]-->
from langchain_community.vectorstores import Lantern
```


### 사용법

랜턴 래퍼에 대한 더 자세한 설명은 [이 노트북](/docs/integrations/vectorstores/lantern)을 참조하세요.
