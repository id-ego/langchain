# Perplexity

> [Perplexity](https://www.perplexity.ai/pro)는 무제한 Pro 검색, 업그레이드된 AI 모델, 무제한 파일 업로드, 이미지 생성 및 API 크레딧을 통해 인터넷을 검색하는 가장 강력한 방법입니다.
> 
> 사용 가능한 [모델 목록](https://docs.perplexity.ai/docs/model-cards)을 확인할 수 있습니다.
