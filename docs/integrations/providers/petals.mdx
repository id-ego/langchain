# Petals

이 페이지는 LangChain 내에서 Petals 생태계를 사용하는 방법을 다룹니다. 설치 및 설정과 특정 Petals 래퍼에 대한 참조의 두 부분으로 나뉩니다.

## 설치 및 설정
- `pip install petals`로 설치
- Hugging Face API 키를 얻고 이를 환경 변수(`HUGGINGFACE_API_KEY`)로 설정

## 래퍼
