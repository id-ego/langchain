# 클라우드플레어

> [클라우드플레어, Inc. (위키백과)](https://en.wikipedia.org/wiki/Cloudflare)는 콘텐츠 전송 네트워크 서비스, 클라우드 사이버 보안, DDoS 완화 및 ICANN 인증 도메인 등록 서비스를 제공하는 미국 회사입니다.

> [클라우드플레어 워커스 AI](https://developers.cloudflare.com/workers-ai/)를 사용하면 REST API를 통해 코드에서 `Cloudflare` 네트워크에서 머신 러닝 모델을 실행할 수 있습니다.
