# 지오판다스

> [GeoPandas](https://geopandas.org/)는 파이썬에서 지리 공간 데이터를 다루는 것을 더 쉽게 하기 위한 오픈 소스 프로젝트입니다. `GeoPandas`는 `pandas`에서 사용하는 데이터 유형을 확장하여 기하학적 유형에 대한 공간 연산을 허용합니다. 기하학적 연산은 `shapely`에 의해 수행됩니다.

## 설치 및 설정

여러 파이썬 패키지를 설치해야 합니다.

```bash
pip install -U sodapy pandas geopandas
```
