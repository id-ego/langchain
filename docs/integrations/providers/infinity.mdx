# 무한대

> [무한대](https://github.com/michaelfeil/infinity)는 텍스트 임베딩을 생성할 수 있게 해줍니다.

## 텍스트 임베딩 모델

무한대 임베딩 모델이 존재하며, 이를 
```python
<!--IMPORTS:[{"imported": "InfinityEmbeddings", "source": "langchain_community.embeddings", "docs": "https://api.python.langchain.com/en/latest/embeddings/langchain_community.embeddings.infinity.InfinityEmbeddings.html", "title": "Infinity"}]-->
from langchain_community.embeddings import InfinityEmbeddings
```
로 접근할 수 있습니다.  
자세한 설명은 [이 노트북](/docs/integrations/text_embedding/infinity)을 참조하세요.
