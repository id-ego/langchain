# 깃

> [깃](https://en.wikipedia.org/wiki/Git)은 소프트웨어 개발 중 프로그래머들이 협력하여 소스 코드를 개발할 때 작업을 조정하기 위해 일반적으로 사용되는 컴퓨터 파일 집합의 변경 사항을 추적하는 분산 버전 관리 시스템입니다.

## 설치 및 설정

먼저 `GitPython` 파이썬 패키지를 설치해야 합니다.

```bash
pip install GitPython
```
