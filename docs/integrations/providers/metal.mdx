# 메탈

이 페이지는 LangChain 내에서 [Metal](https://getmetal.io)을 사용하는 방법을 다룹니다.

## 메탈이란?

메탈은 프로덕션을 위해 구축된 관리형 검색 및 메모리 플랫폼입니다. 데이터를 `Metal`에 쉽게 인덱싱하고, 이에 대한 의미론적 검색 및 검색을 실행할 수 있습니다.

![메탈 대시보드의 스크린샷으로 샘플 데이터와 함께 Browse Index 기능을 보여줍니다.](/img/MetalDash.png "메탈 대시보드 인터페이스")

## 빠른 시작

[메탈 계정 만들기](https://app.getmetal.io/signup)로 시작하세요.

그런 다음, `MetalRetriever` 클래스를 쉽게 활용하여 의미론적 검색, 프롬프트 컨텍스트 등을 위해 데이터를 검색할 수 있습니다. 이 클래스는 `Metal` 인스턴스와 메탈 API에 전달할 매개변수의 사전을 사용합니다.

```python
<!--IMPORTS:[{"imported": "MetalRetriever", "source": "langchain.retrievers", "docs": "https://api.python.langchain.com/en/latest/retrievers/langchain_community.retrievers.metal.MetalRetriever.html", "title": "Metal"}]-->
from langchain.retrievers import MetalRetriever
from metal_sdk.metal import Metal


metal = Metal("API_KEY", "CLIENT_ID", "INDEX_ID");
retriever = MetalRetriever(metal, params={"limit": 2})

docs = retriever.invoke("search term")
```
