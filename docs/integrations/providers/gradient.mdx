# 그래디언트

> [그래디언트](https://gradient.ai/)는 간단한 웹 API를 통해 LLM을 미세 조정하고 완성을 얻을 수 있게 해줍니다.

## 설치 및 설정
- Python SDK 설치:
```bash
pip install gradientai
```

[그래디언트 액세스 토큰과 작업 공간](https://gradient.ai/)을 얻고 이를 환경 변수(`Gradient_ACCESS_TOKEN`)와 (`GRADIENT_WORKSPACE_ID`)로 설정하세요.

## LLM

그래디언트 LLM 래퍼가 존재하며, 이를 통해 접근할 수 있습니다.
[사용 예제](/docs/integrations/llms/gradient)를 참조하세요.

```python
<!--IMPORTS:[{"imported": "GradientLLM", "source": "langchain_community.llms", "docs": "https://api.python.langchain.com/en/latest/llms/langchain_community.llms.gradient_ai.GradientLLM.html", "title": "Gradient"}]-->
```
