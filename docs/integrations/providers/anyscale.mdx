# Anyscale

> [Anyscale](https://www.anyscale.com)는 프로덕션 준비가 완료된 API를 통해 LLM을 실행하고, 조정하며, 확장할 수 있는 플랫폼입니다.  
[Anyscale Endpoints](https://docs.anyscale.com/endpoints/overview)는 많은 오픈 소스 모델을 비용 효율적인 방법으로 제공합니다.

`Anyscale`은 또한 고급 채팅 에이전트를 위해 `LangChain`을 `Anyscale`과 설정하는 방법에 대한 [예제](https://docs.anyscale.com/endpoints/model-serving/examples/langchain-integration)를 제공합니다.

## 설치 및 설정

- Anyscale 서비스 URL, 경로 및 API 키를 가져와 환경 변수(`ANYSCALE_SERVICE_URL`, `ANYSCALE_SERVICE_ROUTE`, `ANYSCALE_SERVICE_TOKEN`)로 설정합니다.  
- 자세한 내용은 [Anyscale 문서](https://www.anyscale.com/get-started)를 참조하십시오.

`openai` 패키지를 설치해야 합니다:

```bash
pip install openai
```
