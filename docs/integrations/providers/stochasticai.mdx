# StochasticAI

이 페이지는 LangChain 내에서 StochasticAI 생태계를 사용하는 방법을 다룹니다. 설치 및 설정, 그리고 특정 StochasticAI 래퍼에 대한 참조로 나뉘어 있습니다.

## 설치 및 설정
- `pip install stochasticx`로 설치
- StochasticAI API 키를 얻고 이를 환경 변수로 설정 (`STOCHASTICAI_API_KEY`)
