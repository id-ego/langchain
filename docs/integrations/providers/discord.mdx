# 디스코드

> [디스코드](https://discord.com/)는 VoIP 및 인스턴트 메시징 소셜 플랫폼입니다. 사용자는 음성 통화, 영상 통화, 문자 메시지, 미디어 및 파일을 개인 채팅 또는 "서버"라는 커뮤니티의 일부로 소통할 수 있습니다. 서버는 초대 링크를 통해 접근할 수 있는 지속적인 채팅룸과 음성 채널의 모음입니다.

## 설치 및 설정

```bash
pip install pandas
```


다음 단계를 따라 `디스코드` 데이터를 다운로드하세요:

1. **사용자 설정**으로 이동합니다.
2. **개인정보 및 안전**으로 이동합니다.
3. **내 데이터 모두 요청**으로 가서 **데이터 요청** 버튼을 클릭합니다.

데이터를 받는 데 30일이 걸릴 수 있습니다. 디스코드에 등록된 이메일 주소로 이메일을 받게 됩니다. 해당 이메일에는 개인 디스코드 데이터를 다운로드할 수 있는 버튼이 포함되어 있습니다.
