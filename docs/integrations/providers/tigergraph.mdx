# TigerGraph

> [TigerGraph](https://www.tigergraph.com/tigergraph-db/)는 본래 분산형 고성능 그래프 데이터베이스입니다.  
그래프 형식의 정점과 엣지로 데이터를 저장하면 풍부한 관계를 형성하여 LLM 응답을 기반으로 하는 데 이상적입니다.

## 설치 및 설정

[TigerGraph 데이터베이스에 연결하는 방법](https://docs.tigergraph.com/pytigergraph/current/getting-started/connection)에 대한 지침을 따르세요.

Python SDK를 설치합니다:

```bash
pip install pyTigerGraph
```
