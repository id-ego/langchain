# ElevenLabs

> [ElevenLabs](https://elevenlabs.io/about)는 콘텐츠를 모든 언어와 음성으로 보편적으로 접근 가능하게 만드는 것을 목표로 하는 음성 AI 연구 및 배포 회사입니다.
> 
> `ElevenLabs`는 29개 언어로 수백 개의 새로운 및 기존 음성으로 음성을 생성할 수 있는 능력을 제공하는 가장 현실적이고 다재다능하며 맥락을 인식하는 AI 오디오를 생성합니다.

## 설치 및 설정

먼저, ElevenLabs 계정을 설정해야 합니다. [여기에서 지침을 따르세요](https://docs.elevenlabs.io/welcome/introduction).

Python 패키지를 설치하세요:

```bash
pip install elevenlabs
```
