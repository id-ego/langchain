# EverNote

> [EverNote](https://evernote.com/)는 사진, 오디오 및 저장된 웹 콘텐츠를 포함할 수 있는 메모를 아카이브하고 생성하기 위해 설계되었습니다. 메모는 가상 "노트북"에 저장되며 태그, 주석, 편집, 검색 및 내보내기가 가능합니다.

## 설치 및 설정

먼저 `lxml` 및 `html2text` 파이썬 패키지를 설치해야 합니다.

```bash
pip install lxml
pip install html2text
```
