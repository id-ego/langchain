# Arcee

> [Arcee](https://www.arcee.ai/about/about-us)는 우리가 SLMs—작고, 전문화된, 안전하며, 확장 가능한 언어 모델이라고 부르는 것의 개발과 발전을 가능하게 합니다.  
SLM 적응 시스템과 원활하고 안전한 통합을 제공함으로써,  
`Arcee`는 기업이 도메인 적응 언어 모델의 잠재력을 최대한 활용할 수 있도록 하여 운영의 혁신적인 변화를 이끌어냅니다.
