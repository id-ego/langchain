# Remembrall

> [Remembrall](https://remembrall.dev/)는 언어 모델에 장기 기억, 검색 증강 생성 및 완전한 관찰 가능성을 제공하는 플랫폼입니다.

## 설치 및 설정

시작하려면 [Remembrall 플랫폼에서 Github로 로그인](https://remembrall.dev/login)하고 [설정 페이지에서 API 키를 복사하세요](https://remembrall.dev/dashboard/settings).

[사용 예제](/docs/integrations/memory/remembrall)를 참조하세요.
