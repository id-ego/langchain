# Label Studio

> [Label Studio](https://labelstud.io/guide/get_started)는 대규모 언어 모델(LLM)을 미세 조정하기 위한 데이터 레이블링에 있어 LangChain에 유연성을 제공하는 오픈 소스 데이터 레이블링 플랫폼입니다. 또한 사용자 정의 훈련 데이터 준비와 인간 피드백을 통한 응답 수집 및 평가를 가능하게 합니다.

## 설치 및 설정

설치 옵션에 대한 [Label Studio 설치 가이드](https://labelstud.io/guide/install)를 참조하십시오.

`label-studio` 및 `label-studio-sdk-python` Python 패키지를 설치해야 합니다:

```bash
pip install label-studio label-studio-sdk
```
