# Beam

> [Beam](https://www.beam.cloud/)는 원격 서버에서 GPU를 사용하여 코드를 실행할 수 있는 클라우드 컴퓨팅 플랫폼입니다.

## 설치 및 설정

- [계정 만들기](https://www.beam.cloud/)
- `curl https://raw.githubusercontent.com/slai-labs/get-beam/main/get-beam.sh -sSfL | sh`를 사용하여 Beam CLI 설치
- `beam configure`로 API 키 등록
- 환경 변수 설정 (`BEAM_CLIENT_ID`) 및 (`BEAM_CLIENT_SECRET`)
- Beam SDK 설치:

```bash
pip install beam-sdk
```
