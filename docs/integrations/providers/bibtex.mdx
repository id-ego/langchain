# BibTeX

> [BibTeX](https://www.ctan.org/pkg/bibtex)는 일반적으로 `LaTeX` 조판과 함께 사용되는 파일 형식 및 참고 문헌 관리 시스템입니다. 이는 학술 및 연구 문서에 대한 서지 정보를 조직하고 저장하는 방법으로 사용됩니다.

## 설치 및 설정

`bibtexparser` 및 `pymupdf` 패키지를 설치해야 합니다.

```bash
pip install bibtexparser pymupdf
```
