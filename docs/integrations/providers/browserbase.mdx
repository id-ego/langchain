# Browserbase

[Browserbase](https://browserbase.com)는 헤드리스 브라우저를 신뢰성 있게 실행, 관리 및 모니터링하기 위한 개발자 플랫폼입니다.

AI 데이터 검색을 다음으로 강화하세요:
- [서버리스 인프라](https://docs.browserbase.com/under-the-hood)로 복잡한 UI에서 데이터를 추출하기 위한 신뢰할 수 있는 브라우저 제공
- [스텔스 모드](https://docs.browserbase.com/features/stealth-mode)로 포함된 지문 인식 전술 및 자동 캡차 해결
- [세션 디버거](https://docs.browserbase.com/features/sessions)로 네트워크 타임라인 및 로그와 함께 브라우저 세션을 검사
- [라이브 디버그](https://docs.browserbase.com/guides/session-debug-connection/browser-remote-control)로 자동화를 신속하게 디버깅

## 설치 및 설정

- [browserbase.com](https://browserbase.com)에서 API 키와 프로젝트 ID를 가져와 환경 변수(`BROWSERBASE_API_KEY`, `BROWSERBASE_PROJECT_ID`)에 설정합니다.
- [Browserbase SDK](http://github.com/browserbase/python-sdk)를 설치합니다:

```python
pip install browserbase
```
