# SemaDB

> [SemaDB](https://semafind.com/)는 번거로움 없는 벡터 유사성 검색 엔진입니다. AI 애플리케이션을 쉽게 구축할 수 있도록 저렴한 클라우드 호스팅 버전을 제공합니다.

SemaDB Cloud, 우리의 호스팅 버전에서는 번거로움이란 포드 크기 계산, 스키마 정의, 파티션 설정, 매개변수 조정, 검색 알고리즘 조정, 복잡한 설치, 복잡한 API가 필요 없음을 의미합니다. [RapidAPI](https://rapidapi.com/semafind-semadb/api/semadb)와 통합되어 투명한 청구, 자동 샤딩 및 대화형 API 놀이터를 제공합니다.

## 설치

필요 없음, [RapidAPI](https://rapidapi.com/semafind-semadb/api/semadb)에서 SemaDB Cloud로 바로 시작하세요.
