# DuckDB

> [DuckDB](https://duckdb.org/)는 프로세스 내 SQL OLAP 데이터베이스 관리 시스템입니다.

## 설치 및 설정

먼저, `duckdb` 파이썬 패키지를 설치해야 합니다.

```bash
pip install duckdb
```
