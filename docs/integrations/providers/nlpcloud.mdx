# NLPCloud

> [NLP Cloud](https://docs.nlpcloud.com/#introduction)는 가장 진보된 AI 엔진을 사용할 수 있도록 해주는 인공지능 플랫폼이며, 자신의 데이터로 엔진을 훈련할 수도 있습니다.

## 설치 및 설정

- `nlpcloud` 패키지를 설치합니다.

```bash
pip install nlpcloud
```


- NLPCloud API 키를 얻고 이를 환경 변수(`NLPCLOUD_API_KEY`)로 설정합니다.
