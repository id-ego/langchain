# 스노우플레이크

> [스노우플레이크](https://www.snowflake.com/)는 대량의 데이터를 저장하고 쿼리할 수 있는 클라우드 기반 데이터 웨어하우징 플랫폼입니다.

이 페이지에서는 `LangChain` 내에서 `Snowflake` 생태계를 사용하는 방법을 다룹니다.

## 임베딩 모델

스노우플레이크는 [Hugging Face](https://huggingface.co/Snowflake/snowflake-arctic-embed-m-v1.5)에서 무료로 오픈 웨이트 `arctic` 임베딩 모델 라인을 제공합니다. 최신 모델인 snowflake-arctic-embed-m-v1.5는 효과적인 벡터 절단을 가능하게 하는 [마트료시카 임베딩](https://arxiv.org/abs/2205.13147) 기능을 제공합니다. 이러한 모델은
[HuggingFaceEmbeddings](/docs/integrations/text_embedding/huggingfacehub) 커넥터를 통해 사용할 수 있습니다:

```shell
pip install langchain-community sentence-transformers
```


```python
<!--IMPORTS:[{"imported": "HuggingFaceEmbeddings", "source": "langchain_huggingface", "docs": "https://api.python.langchain.com/en/latest/embeddings/langchain_huggingface.embeddings.huggingface.HuggingFaceEmbeddings.html", "title": "Snowflake"}]-->
from langchain_huggingface import HuggingFaceEmbeddings

model = HuggingFaceEmbeddings(model_name="snowflake/arctic-embed-m-v1.5")
```
