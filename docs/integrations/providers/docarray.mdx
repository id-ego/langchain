# DocArray

> [DocArray](https://docarray.jina.ai/)는 텍스트, 이미지, 오디오, 비디오, 3D 메쉬 등과 같은 중첩된 비구조적 다중 모드 데이터를 전송하는 라이브러리입니다. 이는 딥러닝 엔지니어가 파이썬 API를 사용하여 다중 모드 데이터를 효율적으로 처리, 임베드, 검색, 추천, 저장 및 전송할 수 있도록 합니다.

## 설치 및 설정

`docarray` 파이썬 패키지를 설치해야 합니다.

```bash
pip install docarray
```


## 벡터 저장소

LangChain은 `DocArray` 라이브러리의 `In-memory` 및 `HNSW` 벡터 저장소에 대한 접근을 제공합니다.

[사용 예제](/docs/integrations/vectorstores/docarray_hnsw)를 참조하세요.

```python
from langchain_community.vectorstores DocArrayHnswSearch
```

[사용 예제](/docs/integrations/vectorstores/docarray_in_memory)를 참조하세요.

```python
from langchain_community.vectorstores DocArrayInMemorySearch
```
