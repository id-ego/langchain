# vlite

이 페이지는 LangChain 내에서 [vlite](https://github.com/sdan/vlite)를 사용하는 방법을 다룹니다. vlite는 임베딩을 저장하고 검색하기 위한 간단하고 빠른 벡터 데이터베이스입니다.

## 설치 및 설정

vlite를 설치하려면 다음 명령어를 실행하세요:

```bash
pip install vlite
```


PDF OCR 지원을 위해 `vlite[ocr]` 추가 기능을 설치하세요:

```bash
pip install vlite[ocr]
```


## VectorStore

vlite는 벡터 데이터베이스 주위에 래퍼를 제공하여 의미 검색 및 예제 선택을 위한 벡터 저장소로 사용할 수 있게 합니다.

vlite 벡터 저장소를 가져오려면:

