# Streamlit

> [Streamlit](https://streamlit.io/)는 데이터 앱을 더 빠르게 구축하고 공유하는 방법입니다.  
`Streamlit`은 데이터 스크립트를 몇 분 안에 공유 가능한 웹 앱으로 변환합니다. 모두 순수 Python으로 작성되었습니다. 프론트엔드 경험이 필요하지 않습니다.  
더 많은 예제를 보려면 [streamlit.io/generative-ai](https://streamlit.io/generative-ai)를 참조하세요.

## 설치 및 설정

`streamlit` Python 패키지를 설치해야 합니다:

```bash
pip install streamlit
```
