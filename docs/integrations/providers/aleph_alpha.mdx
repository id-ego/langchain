# Aleph Alpha

> [Aleph Alpha](https://docs.aleph-alpha.com/)는 2019년에 강력한 AI 시대를 위한 기초 기술을 연구하고 구축하는 사명을 가지고 설립되었습니다. 국제적인 과학자, 엔지니어 및 혁신가 팀이 대규모 언어 모델 및 다중 모달 모델과 같은 변혁적 AI를 연구, 개발 및 배포하며, 가장 빠른 유럽 상업 AI 클러스터를 운영합니다.

> [Luminous 시리즈](https://docs.aleph-alpha.com/docs/introduction/luminous/)는 대규모 언어 모델의 가족입니다.

## 설치 및 설정

```bash
pip install aleph-alpha-client
```


새로운 토큰을 생성해야 합니다. [지침](https://docs.aleph-alpha.com/docs/account/#create-a-new-token)을 참조하세요.

```python
from getpass import getpass

ALEPH_ALPHA_API_KEY = getpass()
```
