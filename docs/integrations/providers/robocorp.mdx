# Robocorp

> [Robocorp](https://robocorp.com/)는 어디서나 어떤 규모에서도 원활하게 실행되는 Python 작업자를 구축하고 운영하는 데 도움을 줍니다.

## 설치 및 설정

`langchain-robocorp` 파이썬 패키지를 설치해야 합니다:

```bash
pip install langchain-robocorp
```


에이전트 애플리케이션에서 통신할 수 있도록 `Action Server`의 실행 인스턴스가 필요합니다. `Action Server`를 설정하고 작업을 만드는 방법은 [Robocorp Quickstart](https://github.com/robocorp/robocorp#quickstart)를 참조하세요.

`Action Server`의 `new` 명령을 사용하여 새 프로젝트를 부트스트랩할 수 있습니다.

```bash
action-server new
cd ./your-project-name
action-server start
```
