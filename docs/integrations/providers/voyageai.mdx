# VoyageAI

VoyageAI와 관련된 모든 기능

> [VoyageAI](https://www.voyageai.com/) Voyage AI는 도메인 및 회사에 맞게 맞춤화된 임베딩 모델을 구축하여 더 나은 검색 품질을 제공합니다.

## 설치 및 설정

통합 패키지를 설치하려면
```bash
pip install langchain-voyageai
```


VoyageAI API 키를 가져와 환경 변수(`VOYAGE_API_KEY`)로 설정합니다.
