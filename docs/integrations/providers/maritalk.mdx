# MariTalk

> [MariTalk](https://www.maritaca.ai/en)는 브라질의 요구를 충족시키기 위해 훈련된 LLM 기반 챗봇입니다.

## 설치 및 설정

MariTalk API 키를 받아야 합니다.

또한 `httpx` Python 패키지를 설치해야 합니다.

```bash
pip install httpx
```
