# Graphsignal

이 페이지에서는 [Graphsignal](https://app.graphsignal.com)을 사용하여 LangChain을 추적하고 모니터링하는 방법을 다룹니다. Graphsignal은 애플리케이션에 대한 전체 가시성을 제공합니다. 체인 및 도구별 지연 시간 분석, 전체 컨텍스트가 포함된 예외, 데이터 모니터링, 컴퓨트/GPU 활용도, OpenAI 비용 분석 등을 제공합니다.

## 설치 및 설정

- `pip install graphsignal`로 Python 라이브러리 설치
- [여기](https://graphsignal.com)에서 무료 Graphsignal 계정 생성
- API 키를 얻고 환경 변수(`GRAPHSIGNAL_API_KEY`)로 설정

## 추적 및 모니터링

Graphsignal은 자동으로 체인을 계측하고 추적 및 모니터링을 시작합니다. 추적 및 메트릭은 귀하의 [Graphsignal 대시보드](https://app.graphsignal.com)에서 사용할 수 있습니다.

배포 이름을 제공하여 트레이서를 초기화합니다:

```python
import graphsignal

graphsignal.configure(deployment='my-langchain-app-prod')
```


추가로 함수나 코드를 추적하려면 데코레이터나 컨텍스트 관리자를 사용할 수 있습니다:

```python
@graphsignal.trace_function
def handle_request():    
    chain.run("some initial text")
```


```python
with graphsignal.start_trace('my-chain'):
    chain.run("some initial text")
```


선택적으로 각 추적에 대한 함수 수준 통계를 기록하기 위해 프로파일링을 활성화합니다.

```python
with graphsignal.start_trace(
        'my-chain', options=graphsignal.TraceOptions(enable_profiling=True)):
    chain.run("some initial text")
```


완전한 설정 지침은 [빠른 시작](https://graphsignal.com/docs/guides/quick-start/) 가이드를 참조하십시오.
