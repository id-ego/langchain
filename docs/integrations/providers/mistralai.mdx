# MistralAI

> [Mistral AI](https://docs.mistral.ai/api/)는 강력한 오픈 소스 모델을 호스팅하는 플랫폼입니다.

## 설치 및 설정

API와 통신하기 위해 유효한 [API 키](https://console.mistral.ai/users/api-keys/)가 필요합니다.

또한 `langchain-mistralai` 패키지가 필요합니다:

```bash
pip install langchain-mistralai
```
