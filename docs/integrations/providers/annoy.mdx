# Annoy

> [Annoy](https://github.com/spotify/annoy) (`Approximate Nearest Neighbors Oh Yeah`)
는 주어진 쿼리 포인트와 가까운 공간의 포인트를 검색하기 위한 Python 바인딩이 있는 C++ 라이브러리입니다. 또한 많은 프로세스가 동일한 데이터를 공유할 수 있도록 메모리에 매핑된 대규모 읽기 전용 파일 기반 데이터 구조를 생성합니다.

## 설치 및 설정

```bash
pip install annoy
```
