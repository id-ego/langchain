# Tair

> [Alibaba Cloud Tair](https://www.alibabacloud.com/help/en/tair/latest/what-is-tair)는 `Alibaba Cloud`에서 개발한 클라우드 네이티브 인메모리 데이터베이스 서비스입니다. 이는 풍부한 데이터 모델과 엔터프라이즈급 기능을 제공하여 실시간 온라인 시나리오를 지원하며 오픈 소스 `Redis`와의 완전한 호환성을 유지합니다. `Tair`는 또한 새로운 비휘발성 메모리(NVM) 저장 매체를 기반으로 하는 지속적인 메모리 최적화 인스턴스를 도입합니다.

## 설치 및 설정

Tair Python SDK 설치:

```bash
pip install tair
```
