# 검색기

[검색기](/docs/concepts/#retrievers)는 비구조화된 쿼리를 기반으로 문서를 반환하는 인터페이스입니다.  
벡터 저장소보다 더 일반적입니다.  
검색기는 문서를 저장할 필요는 없으며, 오직 문서를 반환(또는 검색)하는 기능만 필요합니다.  
검색기는 벡터 저장소에서 생성할 수 있지만, [위키피디아 검색](/docs/integrations/retrievers/wikipedia/) 및 [아마존 켄드라](/docs/integrations/retrievers/amazon_kendra_retriever/)를 포함할 만큼 넓습니다.

검색기는 문자열 쿼리를 입력으로 받아 [문서](https://api.python.langchain.com/en/latest/documents/langchain_core.documents.base.Document.html) 목록을 출력으로 반환합니다.

검색기를 사용하는 방법에 대한 구체적인 내용은 [관련 사용 안내서 여기](/docs/how_to/#retrievers)를 참조하십시오.

모든 [벡터 저장소](/docs/concepts/#vector-stores)는 [검색기로 변환될 수 있습니다](/docs/how_to/vectorstore_retriever/).  
사용 가능한 벡터 저장소에 대한 정보는 벡터 저장소 [통합 문서](/docs/integrations/vectorstores/)를 참조하십시오.  
이 페이지는 [BaseRetriever](/docs/how_to/custom_retriever/)를 서브클래싱하여 구현된 사용자 정의 검색기를 나열합니다.
