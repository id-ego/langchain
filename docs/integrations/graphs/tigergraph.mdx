# TigerGraph

> [TigerGraph](https://www.tigergraph.com/tigergraph-db/)는 본래 분산형 고성능 그래프 데이터베이스입니다.  
데이터를 정점과 간선의 그래프 형식으로 저장하면 풍부한 관계를 형성할 수 있어, LLM 응답을 기반으로 하는 데 이상적입니다.

`TigerGraph`와 `LangChain` 통합의 큰 예시는 [여기에서 제시됩니다](https://github.com/tigergraph/graph-ml-notebooks/blob/main/applications/large_language_models/TigerGraph_LangChain_Demo.ipynb).

## 설치 및 설정

[TigerGraph 데이터베이스에 연결하는 방법](https://docs.tigergraph.com/pytigergraph/current/getting-started/connection) 지침을 따르세요.

Python SDK를 설치하세요:

```bash
pip install pyTigerGraph
```


## 예제

`TigerGraph InquiryAI` 기능을 활용하려면 `langchain_community.graphs`에서 `TigerGraph`를 가져올 수 있습니다.

```python
<!--IMPORTS:[{"imported": "TigerGraph", "source": "langchain_community.graphs", "docs": "https://api.python.langchain.com/en/latest/graphs/langchain_community.graphs.tigergraph_graph.TigerGraph.html", "title": "TigerGraph"}]-->
import pyTigerGraph as tg

conn = tg.TigerGraphConnection(host="DATABASE_HOST_HERE", graphname="GRAPH_NAME_HERE", username="USERNAME_HERE", password="PASSWORD_HERE")

### ==== CONFIGURE INQUIRYAI HOST ====
conn.ai.configureInquiryAIHost("INQUIRYAI_HOST_HERE")

from langchain_community.graphs import TigerGraph

graph = TigerGraph(conn)
result = graph.query("How many servers are there?")
print(result)
```
