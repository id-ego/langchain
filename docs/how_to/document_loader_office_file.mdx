# Microsoft Office 파일 로드하는 방법

[Microsoft Office](https://www.office.com/) 생산성 소프트웨어 모음에는 Microsoft Word, Microsoft Excel, Microsoft PowerPoint, Microsoft Outlook 및 Microsoft OneNote가 포함됩니다. 이는 Microsoft Windows 및 macOS 운영 체제에서 사용할 수 있습니다. 또한 Android 및 iOS에서도 사용할 수 있습니다.

이 문서는 `DOCX`, `XLSX` 및 `PPTX` 문서와 같은 일반적으로 사용되는 파일 형식을 LangChain
[Document](https://api.python.langchain.com/en/latest/documents/langchain_core.documents.base.Document.html#langchain_core.documents.base.Document)
객체로 로드하는 방법을 다룹니다. 이 객체는 이후에 사용할 수 있습니다.

## AzureAIDocumentIntelligenceLoader로 DOCX, XLSX, PPTX 로드하기

[Azure AI Document Intelligence](https://aka.ms/doc-intelligence) (이전 이름: `Azure Form Recognizer`)는 디지털 또는 스캔된 PDF, 이미지, Office 및 HTML 파일에서 텍스트(손글씨 포함), 표, 문서 구조(예: 제목, 섹션 제목 등) 및 키-값 쌍을 추출하는 기계 학습 기반 서비스입니다. Document Intelligence는 `PDF`, `JPEG/JPG`, `PNG`, `BMP`, `TIFF`, `HEIF`, `DOCX`, `XLSX`, `PPTX` 및 `HTML`을 지원합니다.

이 [현재 구현](https://aka.ms/di-langchain)은 `Document Intelligence`를 사용하는 로더로, 콘텐츠를 페이지별로 통합하고 LangChain 문서로 변환할 수 있습니다. 기본 출력 형식은 마크다운이며, 이는 `MarkdownHeaderTextSplitter`와 쉽게 연결되어 의미론적 문서 청크로 나눌 수 있습니다. 또한 `mode="single"` 또는 `mode="page"`를 사용하여 단일 페이지 또는 페이지별로 분할된 순수 텍스트를 반환할 수 있습니다.

### 전제 조건

**East US**, **West US2**, **West Europe** 중 하나의 3개 미리 보기 지역에 Azure AI Document Intelligence 리소스가 필요합니다. 리소스가 없는 경우 [이 문서](https://learn.microsoft.com/azure/ai-services/document-intelligence/create-document-intelligence-resource?view=doc-intel-4.0.0)를 따라 생성하세요. `<endpoint>` 및 `<key>`를 로더의 매개변수로 전달해야 합니다.

```python
<!--IMPORTS:[{"imported": "AzureAIDocumentIntelligenceLoader", "source": "langchain_community.document_loaders", "docs": "https://api.python.langchain.com/en/latest/document_loaders/langchain_community.document_loaders.doc_intelligence.AzureAIDocumentIntelligenceLoader.html", "title": "How to load Microsoft Office files"}]-->
%pip install --upgrade --quiet  langchain langchain-community azure-ai-documentintelligence

from langchain_community.document_loaders import AzureAIDocumentIntelligenceLoader

file_path = "<filepath>"
endpoint = "<endpoint>"
key = "<key>"
loader = AzureAIDocumentIntelligenceLoader(
    api_endpoint=endpoint, api_key=key, file_path=file_path, api_model="prebuilt-layout"
)

documents = loader.load()
```
