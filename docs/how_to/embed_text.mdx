# 텍스트 임베딩 모델

:::info
텍스트 임베딩 모델 제공업체와의 내장 통합에 대한 문서는 [Integrations](/docs/integrations/text_embedding/)를 참조하세요.
:::

Embeddings 클래스는 텍스트 임베딩 모델과 인터페이스하기 위해 설계된 클래스입니다. 많은 임베딩 모델 제공업체(OpenAI, Cohere, Hugging Face 등)가 있으며, 이 클래스는 이들 모두에 대한 표준 인터페이스를 제공하기 위해 설계되었습니다.

Embeddings는 텍스트 조각의 벡터 표현을 생성합니다. 이는 벡터 공간에서 텍스트를 생각할 수 있게 해주며, 벡터 공간에서 가장 유사한 텍스트 조각을 찾는 의미 검색과 같은 작업을 수행할 수 있습니다.

LangChain의 기본 Embeddings 클래스는 두 가지 메서드를 제공합니다: 하나는 문서를 임베딩하는 메서드이고, 다른 하나는 쿼리를 임베딩하는 메서드입니다. 전자는 `.embed_documents`로 여러 텍스트를 입력으로 받으며, 후자는 `.embed_query`로 단일 텍스트를 입력으로 받습니다. 이 두 가지를 별도의 메서드로 제공하는 이유는 일부 임베딩 제공업체가 문서(검색할 내용)와 쿼리(검색 쿼리 자체)에 대해 서로 다른 임베딩 방법을 가지고 있기 때문입니다. `.embed_query`는 부동 소수점 리스트를 반환하며, `.embed_documents`는 부동 소수점 리스트의 리스트를 반환합니다.

```bash

```

### `embed_documents`
#### 텍스트 목록 임베딩

`.embed_documents`를 사용하여 문자열 목록을 임베딩하고 임베딩 목록을 복구합니다:

```python
embeddings = embeddings_model.embed_documents(
    [
        "Hi there!",
        "Oh, hello!",
        "What's your name?",
        "My friends call me World",
        "Hello World!"
    ]
)
len(embeddings), len(embeddings[0])
```


```
(5, 1536)
```


### `embed_query`
#### 단일 쿼리 임베딩
`.embed_query`를 사용하여 단일 텍스트 조각을 임베딩합니다(예: 다른 임베딩된 텍스트 조각과 비교하기 위해).

```python
embedded_query = embeddings_model.embed_query("What was the name mentioned in the conversation?")
embedded_query[:5]
```
