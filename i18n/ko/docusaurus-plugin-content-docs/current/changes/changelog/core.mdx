# langchain-core

## 0.1.x

#### 사용 중단

- `BaseChatModel` 메서드 `__call__`, `call_as_llm`, `predict`, `predict_messages`. 0.2.0에서 제거될 예정입니다. 대신 `BaseChatModel.invoke`를 사용하세요.
- `BaseChatModel` 메서드 `apredict`, `apredict_messages`. 0.2.0에서 제거될 예정입니다. 대신 `BaseChatModel.ainvoke`를 사용하세요.
- `BaseLLM` 메서드 `__call__`, `predict`, `predict_messages`. 0.2.0에서 제거될 예정입니다. 대신 `BaseLLM.invoke`를 사용하세요.
- `BaseLLM` 메서드 `apredict`, `apredict_messages`. 0.2.0에서 제거될 예정입니다. 대신 `BaseLLM.ainvoke`를 사용하세요.
